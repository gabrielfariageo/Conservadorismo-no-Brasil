# Mapas do Projeto

Esta pasta contém os produtos cartográficos gerados durante o desenvolvimento do projeto Conservadorismo no Brasil.

## Produtos Disponíveis

- Resultado das Eleições Presidenciais de 2022 por município.
- Mapas produzidos com QGIS.
- Dados processados em Python.
- Base cartográfica municipal do IBGE.

Novos mapas e análises serão adicionados conforme a evolução do projeto.
